Bounce: 2D game in which the player controls a ball to reach the end

   Febuary 10, 2017       Arthur Tran

Previous Next

About

Bounce is a 2D game I made in java that is essentially a game in which the player manipulates the ball's direction to guide it to the end of the level. As of now, the game has 5 different levels that you can play through.

Features

  • Multiple Levels
  • Different Ball Skins
  • Custom Soundtrack
  • Simple Controls

Releases

v0.0.2 (3-27-18)

  • Added music and various sound effects with the ball. End screen was also implemented.

v0.0.1 (2-10-18)

  • The initial release with five basic levels to play through
